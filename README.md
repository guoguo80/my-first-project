# 气象水文数据分析系统 v1.0.0

## 功能特点

1. **模块化标签页设计**：所有功能以标签页形式组织，操作更直观
2. **五大核心功能**：
   - 数据到报率分析
   - 异常数据分析
   - 降雨数据分析
   - 气象数据分析
   - 设备偏差分析
3. **实时日志系统**：运行日志实时显示在右侧窗口，并自动保存到文件
4. **用户自定义配置**：每个分析功能都可自定义输入输出路径
5. **灵活的到报率分析**：支持多种上传频率选择（1分钟、5分钟、10分钟、30分钟、60分钟、自定义）
